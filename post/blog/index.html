基于hugo搭建个人博客

1.hugo下载

  • [github搜索hugo,下载对应版本的二进制文件，zip结尾https://github.com/gohugoio/hugo]

  • [解压到本地（无需安装），将存放的路径添加到 环境变量 中的 系统变量]

  • [打开cmd 输入hugo version，查看是否成功]

  • [成功后，打开cmd(最好是管理员模式),输入 hugo new site myblog]

2.git下载

  • [官网下载对应安装包，然后进行安装，安装过程全部默认，安装路径自选https://git-scm.com/downloads]

  • [安装完后打开cmd,输入git,如果成功，则会出现usage:……………..,反之，去环境变量中的系统变量中的path,查看是否配置]

  • [成功显示后 cmd中输入 git config –global user.name “xxx” 有意义的名字，如你github上的用户名 git config –global user.email “xxx” 有意义的名字，如你github上的邮箱 ]

3.选择喜欢的博客模版

  • [登录https://themes.gohugo.io]

  • [选择适合自己的模版，会有一条 git … .git的命令，复制]

  • [cmd到你myblog的根目录,这边根目录如： C:\Windows\System32\myblog\themes]

  • [粘贴 回车]

  • [cmd到 C:\Windows\System32\myblog]

  • [输入 hugo server -t 主题名 –buildDrafts]

  • [访问http://localhost:1313，查看是否成功]
